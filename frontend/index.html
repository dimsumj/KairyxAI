Data Sandbox

Manual Field Mapping

Override raw field paths when a connector payload uses mismatched or duplicate field names before normalization.

Example mapping keys: canonical_user_id, event_name, event_time, source_event_id, campaign, adset, media_source. Values are raw JSON paths like event_properties.PID.

Example mapping JSON 
{
  "canonical_user_id": "event_properties.PID",
  "event_name": "event_type",
  "event_time": "event_time",
  "source_event_id": "event_properties.insert_id",
  "campaign": "event_properties.campaign_name",
  "adset": "event_properties.adgroup_name",
  "media_source": "event_properties.network"
}
